Online Learning Platform (Django)

Description

This is a web-based Online Learning Platform developed using Django. The application allows users to browse courses, enroll in them, and make secure payments. It provides an easy and interactive way for learners to access educational content anytime.

Features

User Registration & Login

Browse Available Courses

Course Enrollment System

Secure Payment Integration (Razorpay)

User Dashboard

Admin Panel for Course Management

Technologies Used

Python

Django

HTML, CSS, JavaScript

SQLite (or MySQL)

Razorpay API

Installation & Setup

Clone the repository

git clone

Navigate to the project folder

cd online-learning

Install dependencies

pip install -r requirements.txt

Run migrations

python manage.py migrate

Start the server

python manage.py runserver

Open in browser

http://127.0.0.1:8000/⁠�
